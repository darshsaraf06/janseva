Janseva is a web application designed to empower citizens to report local civic issues directly to government authorities. This project aims to bridge the communication gap between the public and government bodies, enabling a more efficient and transparent process for resolving community problems.

The application allows users to report issues such as:
Potholes
Broken streetlights
Water leaks
Garbage and sanitation issues
Damaged public property

Key Features
1)User-friendly Interface: A simple and intuitive design built with HTML and CSS for easy navigation.

2)Issue Reporting: Citizens can submit detailed reports of problems, including location and a brief description.

3)Backend Logic: The core functionality is powered by Java, which handles user requests, data processing, and interactions with the database.

4)Data Management: MySQL is used to store and manage all user information and reported issues securely.

Technologies Used
Frontend: HTML, CSS
Backend: Java
Database: MySQL
